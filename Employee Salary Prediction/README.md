# Employee Salary Prediction

A simple Machine Learning project that predicts an employee's salary based on years of experience using **Linear Regression**.

## Features

- Predicts salary based on years of experience
- Built with Python and Scikit-learn
- Beginner-friendly project
- Command-line interface

## Technologies Used

- Python
- NumPy
- Scikit-learn

## Installation

Install the required libraries:

```bash
pip install numpy scikit-learn
```

## Run the Project

```bash
python employee_salary_prediction.py
```

## Example

**Input:**
```
Enter years of experience: 5
```

**Output:**
```
Predicted Salary: Rs. 50000
```
